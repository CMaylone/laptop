# laptop
Quick set-up scripts for new laptops

## Getting Started

### Debian

```
wget https://raw.githubusercontent.comMaylone/laptop/master/debian.sh -O - | sudo -E bash -
```

## What is installed?

* git
* terminator
* visual studio code
* chrome
* inconsolata font
* node

## What can be set-up?

* git global name and email
* intial ssh keygen
